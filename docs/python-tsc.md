# Python Thresholded Subspace Clustering (TSC)

This page documents the Python implementation of the **Thresholded Subspace Clustering (TSC)** algorithm.
The main implementation lives in `TSC_Python/tsc.py`
An additional MLX-based implementation is available in `TSC_Python/tsc_mlx.py` for Apple Silicon acceleration.
